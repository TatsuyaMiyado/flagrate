Object-Oriented

Write the good code. More robust programming. 

var modal = flagrate.createModal({
  title  : 'Hey',
  text   : 'I want to eat pilaf...!'
});

modal.open();
// These are equivalent. Do you want to use the new yet?
var notify = flagrate.createNotify();

Pure JavaScript

No dependencies, No pollution.
Works with any web framework. 

var popover = flagrate.createPopover({
  text: 'Yay!'
});

// Pure
popover.open(document.getElementById('x'));
// jQuery
popover.open($('#x')[0]);
// Prototype.js
popover.open($('x'));
